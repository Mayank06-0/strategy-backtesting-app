## 📌 Overview

This project is a **Python-based trading strategy backtesting system** deployed as a web application using Streamlit.

It allows users to test trading strategies on historical stock data and evaluate performance.

---

## ⚙️ Features

* 📈 Moving Average Crossover Strategy (Golden Cross / Death Cross)
* 📊 Real-time backtesting on historical data (Yahoo Finance)
* 💰 Portfolio simulation with capital tracking
* 📉 Performance metrics:

  * Total Return
  * Win Rate
  * Max Drawdown
  * Sharpe Ratio
* 📊 Interactive visualization:

  * Price chart with signals
  * Volume
  * Equity curve vs Buy & Hold

---

## 🛠️ Tech Stack

* Python
* pandas, numpy
* matplotlib
* yfinance
* Streamlit

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
